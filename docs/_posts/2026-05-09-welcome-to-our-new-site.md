We are pleased to announce the launch of our new website. As a non-profit organization focused on Yoga, we wanted a space that reflects tranquility, simplicity, and focus.

### How to Add New Articles

This site uses Jekyll, which means adding a new article is as simple as creating a new file!

To publish a new article:
1. Create a new markdown file in the `docs/_posts` directory.
2. Name the file using the format: `YYYY-MM-DD-title-of-your-article.md`.
3. Add the YAML front matter at the top of the file (like the one used for this post).
4. Write your content in Markdown below the front matter.

Your new article will automatically appear on the Articles page.
